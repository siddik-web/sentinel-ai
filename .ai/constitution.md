# Sentinel AI — Constitution

The constitution defines the non-negotiable rules that govern this project. Every
decision, prompt, workflow, and line of code must comply with what is written here.
When a conflict arises, the constitution wins.

## Core Tenets

1. **Correctness over speed.** Ship behavior that is verifiably right before it is fast.
2. **Transparency.** Decisions are recorded in `.ai/decisions/`; nothing important is implicit.
3. **Safety by default.** Destructive or outward-facing actions require explicit confirmation.
4. **Human in the loop.** The system advises and executes, but accountability stays with people.
